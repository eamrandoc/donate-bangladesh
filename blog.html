What is the Document Object Model (DOM)?

Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects and manipulate HTML dynamically.

    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is a technique in the DOM where a single event listener is attached to a parent element, capturing events from child elements as they bubble up. This is useful because It improves performance by reducing the number of listeners, simplifies management of dynamic content, and enhances memory efficiency by requiring fewer resources for event handling.

    How do you manipulate an element's attributes and styles using the DOM?

    We can manipulate an element's attributes and styles in the DOM using JavaScript. To change attributes, we use methods like setAttribute, getAttribute, and removeAttribute. This allows for dynamic updates to elements. For instance, you could set a background color, change text size, or toggle classes based on user interaction, enhancing the user experience effectively.
